General settings tab

This tab represents the most basic settings of the emulated machine. Here you will be able to choose the macintosh model to be emulated, the ROM file to be used and the amount of memory. You can also change the name of the virtual machine to anything you like.

Name

In this field you can change the virtual machine's name. If you type "return" on your keyboard, the new name will be automatically saved.

Rom file

You can set a previously saved ROM image file associate a new ROM image file by clicking "open" and choosing the file or using drag & drop from a Finder window to drop the file into the icon area. If a ROM file is chosen, it won't become empty again unless the chosen file is removed using the Assets Window. The reason for that is that the virtual machine cannot work without a ROM image file and removing the file renders the machine useless.

Any limitations or important details of the chosen ROM image will be displayed in the comments area.

Model

This field is where you can choose a macintosh model for this virtual machine. Please note that only a few models will be available depending on the ROM image used. The default value is 'Quadra 900'.

This field is disabled for SheepShaver virtual machines, because SheepShaver emulates only one model, which is the Power Macintosh 9500.

Memory

This field defined the memory the emulator will be allowed to use for the virtual machine. The legacy Macintosh emulators usually won't use more than 2GB, and that's why there are no further options. Different values can be edited manually into the text field with the number.

The default value is 32.
