Network settings tab

The network tab allows you to give the virtual machine the ability to access a folder shared with the host machine and to use the local network driver.

Enable ethernet

This simply allows the virtual machine to access the ethernet.

Folder path

This is the path to the folder that will be shared between the host and the guest systems. The standard path is the user's home folder. A different path can be used by clicking in the "search" button.

Using tap0

This is an advanced setting for users that want to use the tap0 network interface instead of OSX's default interface slirp.

This is used to enable Appletalk in the latest releases of OSX. However, this driver must be installed manually for this to work.

If you don't know if you need this, just leave the setting turned off.

UDP settings

This is an advanced setting for users that want to use User Datagram Protocol (UDP). If you don't know if you need this, just leave the setting turned off.
