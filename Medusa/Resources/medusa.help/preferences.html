Preferences

The preferences window is divided into three tabs:

General tab

There are two options in the general tab: the first one is a flag to open the settings of a newly created virtual machine; the second limits Basilisk II's macintosh model to Quadra 900 and Mac IIci, which are the "officially supported" models for emulation.

Share tab

Here you will find a field to define the default shared folder address that will be used if the option is chosen at the network area of the settings window of each virtual machine.

Advanced tab

At this tab there are three cheboxes. The first one, if unticked, will open the Assets Manager every time Medusa executes. The second option enables extra controls in the application. The third checkbox allows you to have the a hidden file generated by Basilisk II called "XPRAM" deleted every time a virtual machine is turned off. The button bellow deletes the same file.

Advanced information and hidden controls

If the hidden controls are to be displayed, there will be four extra buttons in the virtual machines window and a couple of extra options in the virtual machine settings.

The four buttons work as follows:

  • The button with a crosshair/reticle icon is meant to kill an virtual machine process.
  • The button with a tick icon generates the virtual machine's preferences file in the Application support directory used by Medusa.
  • The button with a magnifier icon locates the preferences file and highlights it on Finder. It silently fails if the file doesn't exist.
  • The button with an "i" icon opens a similar window, synchronised with the virtual machine's list and extra information like the unique ID of the virtual machine, the status and process id.
