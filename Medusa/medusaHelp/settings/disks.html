Disk settings tab

In this tab you can manage the disk images that will be used by your virtual machine. There are two lists of disk images: the left one displays all disks available in Medusa and the right one all the disks used by this virtual machine.

To use an available disk, select the desired disk in the left and click in the arrow button below the list. The disk's name will appear in the right list in a single row. To remove the disk image from the second list, select it and click the 'minus' button below the list. This will not affect the files in the left pane. To edit the left pane items, you must use the Assets Manager.

A disk image can also be dropped from Finder into the right list to be added to the virtual machine.

Important: The order of the disk images is important. The upper-most disk will be used by the emulator to boot the virtual machine. If the disk at the top is not bootable, the emulator will boot from the next volume in the list. You can change the order of the disks by dragging and dropping them into position.
